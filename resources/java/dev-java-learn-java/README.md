## Why this resource is useful

Dev.java is useful for learning and reviewing:

- Java language basics
- setting up the JDK
- writing first Java applications
- JShell
- object-oriented Java basics
- collections
- streams
- modern Java features
- IDE-oriented workflows with IntelliJ IDEA, VS Code and Eclipse

This resource complements the formal JDK documentation with practical learning material.
