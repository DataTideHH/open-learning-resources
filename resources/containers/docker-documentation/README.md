Docker documentation is useful for learning and reviewing:

- container basics
- images and containers
- Docker Engine
- Docker CLI basics
- Docker Compose
- Dockerfiles

- volumes
- networks
- reproducible development environments
- database and service containers for local learning projects

For this portfolio, Docker is especially relevant as a tool for reproducible SQL, database and development environments.
