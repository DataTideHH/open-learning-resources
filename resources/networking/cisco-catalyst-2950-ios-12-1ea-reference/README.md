This is a link-only reference entry.

No local copy of Cisco documentation is included in this repository.

The Cisco Catalyst 2950 is legacy hardware. In this repository it is documented as a historical Layer-2 lab switch for VLAN, trunking, spanning tree and basic Cisco IOS command-line practice.

## Why this resource is useful

This reference is useful for lab work with an older Cisco Catalyst 2950 switch, especially for:

- Cisco IOS CLI basics
- switch configuration commands
- VLAN-related commands
- trunking-related commands
- spanning tree commands
- interface configuration
- show and debug commands
- understanding legacy Cisco Catalyst behavior
