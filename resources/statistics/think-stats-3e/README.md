## Important note

The third edition of Think Stats is notebook-first.

This local file is not a PDF or EPUB. It is the official notebook ZIP without solutions from the official source repository, pinned to the current v3 branch commit at the time it was added.

## Why this resource is useful

Think Stats is useful for learning and reviewing:

- exploratory data analysis
- distributions
- probability mass functions
- cumulative distribution functions
- probability density functions
- relationships between variables
- estimation
- hypothesis testing
- least squares
- multiple regression
- time series analysis
- survival analysis
- practical statistics with Python notebooks
