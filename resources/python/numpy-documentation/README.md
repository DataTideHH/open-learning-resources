Reason for inclusion:

NumPy is a core Python library for numerical and scientific computing. It is relevant for array-based data processing, numerical operations, statistical calculations, linear algebra and as a foundation for many Python data libraries.

This reference supports learning around Python data workflows, pandas, matplotlib, scikit-learn and reproducible data analysis projects.

Reason for link-only:

NumPy documentation is dynamic project documentation. A local copy would become outdated over time and should not be mirrored in this repository.
