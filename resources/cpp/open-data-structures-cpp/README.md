## Why this resource is useful

Open Data Structures in C++ is useful for learning and reviewing:

- asymptotic analysis
- arrays and array-based lists
- linked lists
- skiplists
- hash tables
- binary trees
- random binary search trees
- red-black trees
- heaps
- sorting algorithms
- graphs
- B-trees
- practical C++ implementations of classic data structures
