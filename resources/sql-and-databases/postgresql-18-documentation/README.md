## Why this resource is useful

PostgreSQL documentation is useful for learning and reviewing:

- relational database concepts
- SQL language features
- data types
- functions and operators
- indexes
- transactions
- concurrency control
- query planning
- administration basics
- backup and restore
- security and privileges
- PostgreSQL-specific behavior compared with other SQL systems
