## Important note

This is a link-only reference entry.

No local copy of MariaDB documentation is included in this repository. The documentation is web-first and current, and a local mirror would become outdated quickly.

## Why this resource is useful

MariaDB documentation is useful for learning and reviewing:

- MariaDB Server usage
- SQL behavior in a MySQL-compatible database
- server management
- users and privileges
- backup and restore
- clients and utilities
- performance and high availability concepts
- practical database lab work

This is especially relevant because MariaDB is part of the current local learning and lab setup.
