# Adding Resources Checklist

Before adding a third-party PDF, book, documentation export or other file to this repository, check:

- Is the material only free to read, or is redistribution explicitly allowed?
- Is the license clearly stated?
- Is commercial use allowed or prohibited?
- Is attribution required?
- Are modifications allowed?
- Are ShareAlike terms required?
- Is the official source linked?
- Is the local copy necessary, or would a link be better?

If redistribution is unclear, do not upload the file. Link to the official source instead.
